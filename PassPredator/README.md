# PassPredator - Ethical Password Cracking Tool
## Features:
- **Brute Force Attack**
- **Dictionary Attack**
- **AI-Powered Mask Attack**
- **Hash Generator**
- **GUI Interface**

## Usage:
1. Run `main.py` to start the GUI.
2. Use different attack methods to test password security.
3. Use `hash_generator.py` to create test hashes.
